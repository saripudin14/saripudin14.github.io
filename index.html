About Me

Nama saya Saripudin, seorang mahasiswa baru dari Jakarta yang saat ini sedang mendalami bidang web development. Saya belajar HTML, CSS, Javascript, dan Bootstrap untuk membangun tampilan web yang menarik dan responsif. Dengan dasar-dasar ini, saya berusaha mengembangkan keterampilan dalam merancang halaman web yang fungsional dan estetik. Saya sangat antusias dalam menjelajahi lebih banyak tentang teknologi web dan senang menghadapi tantangan untuk memperluas pengetahuan saya di bidang ini.
